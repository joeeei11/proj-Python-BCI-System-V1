# proj-Python-BCI-System-V1

**NeuroPilot V1 —— 脑机接口系统初代版本，包含完整的 EEG 采集、处理与设备控制模块。**

## 功能特性

- EEG 实时采集与预处理
- CSP 特征提取与运动想象分类
- 设备控制与调试界面
- 受试者数据管理
- PyCharm + Lingma AI 辅助开发环境

## 快速开始

### 环境要求

- Python >= 3.10
- numpy, scipy, scikit-learn, PyQt5

### 安装步骤

```bash
git clone https://github.com/joeeei11/proj-Python-BCI-System-V1.git
cd proj-Python-BCI-System-V1
pip install numpy scipy scikit-learn PyQt5
```

### 基础用法

```bash
python main.py
```
